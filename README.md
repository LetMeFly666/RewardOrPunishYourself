# RewardOrPunishYourself

奖惩自己，如果做得好就奖励自己金币，做得不好（没管住自己看了1h的番）扣除自己的金币。让奖励变成习惯。

简单小[脚本](PythonScript/main.py)，记录自己今日的自律
